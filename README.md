# Modern C++チャレンジ問題67のたったひとつの冴えた解き方
Modern C++チャレンジ読書会 vol.27

## 実装コメント

回答ではデコレータを使っていたが、ストラテジーパターンを採用した。
バリデーションの結果、エラー理由がわからなければバリデーションとして三流。
ということで、自分で作ったresult型のライブラリを投入して真面目にコーディングした。
